## 📽️ 프로젝트 소개
관심 영화의 (재)개봉 예정 정보를 자동으로 수집·분석해 알림으로 제공하는 서비스

### 배경
- 2025년 기준 OTT 이용자 수 증가(3년간 약 21%)와 함께 극장 관객 수 전년 대비 32.5% 감소로 영화관 산업 위축
- 극장 재개봉 영화 정보가 영화관별로 분산되어 있어 사용자가 직접 확인해야 하는 불편함 존재

### 목적
- 관심 영화 알림으로 정보 탐색 부담을 줄이고 사용자 편의성 향상
- 극장 관람 유도를 통해 몰입감 있는 관람 경험 확대 및 영화 산업 활성화 기여

### 기간 및 인원
- 2025.02 ~ 2025.04 / 1인 프로젝트
